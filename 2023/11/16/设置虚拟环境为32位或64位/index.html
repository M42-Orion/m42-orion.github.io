设置虚拟环境为32位或64位

Created2023-11-16|Updated2024-04-07|学习
|Post Views:

检查或更改conda设置

通过anaconda prompt，输入 conda info ，显示conda信息并检查平台

检查Python环境

prompt中输入python，检查当前环境中的Python是64位还是32位,” MSC v.1916 32位”为32位，” MSC v.1916 64位”为64位。

切换到64位平台及环境

  • 第一步：set CONDA_FORCE_32BIT=0
  • 第二步：set CONDA_SUBDIR=win-64
  • 第三步：conda info 查看结果

创建虚拟环境来安装64位的python

prompt中输入conda create -n python39_64 python=3.9

激活虚拟环境

conda activate python39_64

可能遇到的问题
如果发现虚拟环境安装的python不是64位的。则需要先执行命令conda deactivate退出虚拟环境，conda remove -n python39_64 –all 删除刚创建的虚拟环境，再试着重新操作（1）（2）步骤下载64位python。

至此，已经完成python32位和64位并存。

退出虚拟环境

执行命令conda deactivate
