标准“鸭子类型”

在典型的Python代码中，许多可以将列表或字典作为参数的函数只需要它们的参数以某种方式“类似列表”或“类似字典”。“类似列表”或“类似字典”（或类似的东西）的特定含义被称为“鸭子类型”，并且在惯用Python中常见的几种鸭子类型是标准化的。
